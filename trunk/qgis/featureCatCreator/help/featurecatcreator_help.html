Fields tab

The form contains the following elements :

Attribute table button
This button shows the attribute table of the current layer
Current field
This combobox selects the layer field you want to define informations of
Reset list
This button re-read the field list, type and definition from the data. It zeroes defined values. If classification is checked, it reloads field informations with a classification, otherwise no classification is done.
Classification
If this checkbox is active, when reading fields informations, extract values from data. Classification parameters are set in the spinboxes just below.
